🐍 Retro Style Snake Game

Retro Style Snake Game is a single-player game developed using Unity and C#, aiming to recreate the nostalgic experience of the classic Snake game.

The player controls a snake to collect food, increase their score, and avoid collisions. The project includes a scoring system, difficulty selection, restart and exit options, and a clean modular design that follows software engineering best practices.


🎮 Features

-Classic Snake gameplay mechanics

-Dynamic score tracking system

-Collision detection (walls & self-collision)

-Restart option

-Exit option

-Modular and maintainable code structure


🛠️ Technologies Used

C#

Unity Game Engine

🧠 Project Objectives

This project was developed to:

Apply core game development principles

Reinforce understanding of the Unity game loop

Implement collision detection systems

Follow modular software design practices

Adhere to software engineering best practices
